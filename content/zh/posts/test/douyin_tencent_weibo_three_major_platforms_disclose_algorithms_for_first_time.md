为什么要分三个阶段呢？

因为算法是要消耗大量算力的，只能先初筛掉一大部分内容，到排序环节再进行精密计算出内容价值

![img](https://i-blog.csdnimg.cn/direct/01b4c140fbf64bb5a71b83c0b78875a4.png)

| 阶段名称           | 作用                                 | 主要技术/方法                                                | 作用                                     | 处理量级              |
| ------------------ | ------------------------------------ | ------------------------------------------------------------ | ---------------------------------------- | --------------------- |
| 第一阶段内容池准入 | 初步筛选新内容，确保合规性和基础质量 | - 规则过滤（如敏感词检测）- 简单模型预测（如点击率预估）     | 快速过滤低质内容保障内容池基础质量       | 全量新内容（亿级/日） |
| 第二阶段召回环节   | 从海量内容中粗筛候选集               | - 双塔召回模型（用户塔 & 内容塔）- 向量化表征匹配（余弦相似度） | 高召回率覆盖潜在兴趣低延迟响应（毫秒级） | 百万级候选内容        |
| 第三阶段排序环节   | 精细化打分并生成最终推荐列表         | - 深度学习模型（如Wide&Deep）- 实时特征工程（用户实时行为）  | 高精度个性化排序动态适应实时反馈         | 千级候选内容          |

深度学习模型结构基础

- 分三层：输入层→隐藏层→输出层（其实是参考人工神经网络的结构）
- 我们常说的“深度学习”中的“深度”，指的就是在神经网络中使用“多层”，通常情况下，层数越多，算法的能力越强。

类 比：抓小偷的过程

- 输入层：保安登记特征（小区门口的保安负责登记来访人员信息，例如戴黑帽子、穿红衣服、背双肩包（像素级基础特征）。登记完成后，他把这些信息写成小纸条，传给下一层侦探）
- 隐藏层：侦探推理（第一组侦探：看小纸条拼拼图，发现“黑帽子+红衣服”常出现在便利店盗窃案；第二组侦探：查历史档案，发现“双肩包+上午8点”对应小区快递被偷案；第三组侦探：画嫌疑人画像，综合“黑帽子、红衣服、双肩包+多次出现在案发地”，锁定惯犯特）
- 输出层：警长决策（警长拿到综合报告，对比数据库后大喊：“这是上周连环盗窃案的A！”）

训练机制

- 模型输出 vs. 目标结果

Wide&Deep模型（抖音应用）

- Wide部分：记忆能力（学习共现频率）：可以被理解为模型直接学习并利用历史数据中物品或者特征的“共现频率”的能力。说人话就是算法记住“如果点击过A,就推荐B”的规则的能力
- Deep部分：泛化能力（特征交叉挖掘）：被理解为模型传递特征的相关性，以及发掘稀疏甚至从未出现过的稀有特征与最终标签相关性的能力。即使是非常稀疏的特征向量输入，也能得到较稳定平滑的推荐概率。说人话就是，对于一些非常小众、冷门的内容，算法都能通过少量的数据找到精准用户推荐。
- 对比传统的协同过滤算法：wide&deep模型解决的问题是“推荐的结果头部效应比较明显，导致造成信息单一问题。”（因为传统的算法在缺少样本量的情况下，对于这些不那么大众的内容就无法找到合适的用户推送）

双塔召回模型

- 用户塔 & 内容塔 → 向量化表征：

1）把用户和内容都转化为数学空间里的一个个点，就像是给用户和内容贴上了独特的“数字标签”，这个过程叫做向量化表征学习（如展示小猫的视频为0，展示小狗的视频为1，短视频为0，长视频为1，那么一个展示小猫的长视频即（0，1），其实际长度取决于特征有多少维度，用户特征同理）

2）将训练过的所有内容表征的向量值和当前用户表征的向量值的距离进行对比，距离越接近代表用户越喜欢。

说人话，就是把“用户+行为”、“内容”换算成二进制的一串数字（例如010010101），这样如果“用户+行为”与“内容”二进制值越接近，预测用户喜欢的可能性越大。

- 向量空间距离匹配（“数字指纹”匹配）

对比

| 对比维度       | 传统推荐算法：协同过滤（CF）                                 | 深度学习推荐算法升级                                         |
| -------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 核心逻辑       | 基于用户行为相似性进行推荐，无需理解内容语义                 | 基于多维特征建模，理解内容与用户的深层关系                   |
| 实现步骤       | - 构建用户-内容共现矩阵- 计算相似度（如余弦相似度）- 推荐相似用户喜好内容 | - 多特征输入深度模型- 训练Wide&Deep或双塔模型- 向量匹配推荐内容 |
| 代表模型       | 协同过滤（UserCF、ItemCF）                                   | Wide&Deep、双塔召回模型                                      |
| 优势/缺点      | 优点1）简单直观，依赖用户行为数据，部署成本低缺点：1）泛化能力差，冷门内容得不到推送，热门大众内容垄断流量，冷启动问题明显2）计算单一，未结合复杂的用户属性和行为计算 | 优点：1）表达能力强，模型灵活，适配复杂业务场景2）泛化能力强，对于一些非常小众、冷门的内容，算法都能通过少量的数据找到精准用户推荐3）“数字标签”缺点：1）算法结构复杂，训练成本高2）解释性较弱 |
| 在抖音应用情况 | 初期用于基础推荐和兴趣匹配                                   | 当前主流架构，结合长期价值预测，支持大规模内容分发           |

## 3.1 微博热搜算法公开信息整理

### 3.1.1 热搜榜升级内容

引入公众性指标，优化榜单排序逻辑

新增“公众性指标”：评估热点覆盖的人群广泛性

目的：更真实反映更广泛用户的关注

排序逻辑调整：

| 原逻辑       | 新逻辑                             |
| ------------ | ---------------------------------- |
| 以热度值为主 | 加入公众关注度，特别优化榜单前十位 |

什么是微博热搜的「热值」？

热值是由基础传播力、质量系数、正能量系数和人工校准组成

评论、阅读和搜索是最关键的

1. 基础传播力（权重最大）

讨论量：有效用户评论数（排除水军、机器号）

阅读量：话题页面的真实用户访问量

搜索量：通过微博搜索该话题的次数

2. 质量系数（权重次之）

传播深度：二次转发率（用户转发后引发的再传播层级）

权威引用：被央媒、政府账号转发的次数（1次=普通账号100次）

有效互动：含实质性观点的评论占比（如超过30字且有话题标签）

3. 正能量系数（权重第三）

议题类型：乡村振兴、科技创新等正向议题获得1.2-1.8倍加成

价值观审核：AI 模型识别内容倾向（负面话题最高仅能获得50%系数）

4. 人工校准

总编辑团队每15分钟调整一次榜单

强制干预机制：娱乐八卦类话题单日最多上榜3条

### 3.1.2 升级管理规则，严控前十内容质量

调整热搜前十的管理细则

不同领域制定差异化标准（社会、文娱、垂直类），确保高位热搜具备：（1）信息质量高；（2）讨论价值强

重大事件发生时：动态调整榜单内容与排序——简单来说，就是有很强的人工干预性质

强调平台社会责任和公众利益优先，因此，优化后的排序逻辑，可能会使部分热点的排名与热度值的绝对排名有所不同

# 思考：Q&A

## 1、为什么中央网要管理算法的问题？真的是单纯为了用户体验吗？

1）改善用户体验：现在的内容太多太杂了，而且AI时代开启，真假难分

2）夺回"算法主权"：国家希望防止资本通过算法无序扩张，也希望未来通过算法影响民众吸收的信息（美国已经有类似的公司在做这种事）

3）价值观培养：像微博那种热搜基本上是明星吃瓜之类的，毫无营养，需要一些正能量的东西

## 2、平台应该如何生存？

1）必须响应央视网的号召：例如抖音的党媒优先推荐通道，就是最好的响应

2）开发合规衍生服务（微博的舆情分析SaaS工具）

## 3、普通人在这次算法调整下，能有什么机会？

- 抖音：通过抖音算法的调整看，普通人可以趁着这次打开自己的曝光，因为算法应该是越来越去中心化，防止头部垄断流量
- 视频号：冷启动要思考社交属性，好友点赞权重占比很高，要想想好友都喜欢什么内容
- 微博：这里应该是专门打击“星八卦”的占比，因为整个微博全是追星和负能量的东西（现在微博上基本都很少熟人社交，大部分是陌生人社交）所以很多人性的阴暗面显示出来。时事政治、科技、正能量等内容会有机会
