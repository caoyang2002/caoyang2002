## 一、非标零部件制造平台

此类平台主要解决小批量、多品种、定制化的金属/非金属零件采购难题，是产品研发与快速验证的关键。

| 平台 | 核心定位 | 适用场景与核心特点 |
| :--- | :--- | :--- |
| **[速加网](https://www.sogaa.net)** | 机械零部件一站式**云制造**平台 | **场景**：产品原型打样、小批量试产、供应链外包。<br>**特点**：整合全国产能，在线报价与订单管理快，强调**交付速度**与**供应链优化**。 |
| **[铨洲智造](https://www.quanzhouzhizao.com)** | **精密金属零部件**垂直自营平台 | **场景**：3C电子、自动化设备、医疗器械中对**精度、表面处理**要求高的零件。<br>**特点**：自营工厂，**质量控制严格**，擅长复杂结构件与高难度工艺。 |
| **[海智在线](https://www.haizol.com)** | 全球非标零部件**产能对接与采购**平台 | **场景**：复杂结构件、大型钣金、模具等“难找供应商”的部件全球寻源。<br>**特点**：**全球供应商网络**，擅长解决“多品种、小批量、高难度”采购，提供供应链管理服务。 |
| **[Xometry (择幂科技)](https://www.xometry.com)** | **全球性按需制造**市场平台 | **场景**：有海外项目、需要本地化供应或对比全球制造成本的项目。<br>**特点**：庞大的国际供应商网络，支持CNC、3D打印、钣金等**即时在线报价**，适合出海企业。 |
| **[Fictiv](https://www.fictiv.com)** | **云端数字制造生态系统** | **场景**：注重**全流程数字化管理**的精密机加工与注塑项目。<br>**特点**：从设计评审、报价、生产到物流全流程在线可视化，项目管理效率高。 |
| **[立谱智造](https://www.leapmanufacturing.com)** | **AI驱动的非标制造**赋能平台 | **场景**：追求快速核价与透明化生产的非标件采购。<br>**特点**：利用**AI大模型快速报价**，提供从设计到交付的全链路监控与数据分析。 |
| **[RapidDirect](https://www.rapiddirect.com)** | 全球化按需**精密制造**平台 | **场景**：对加工精度（如航空航天、光学部件）要求极高的定制件采购。<br>**特点**：提供从定制、半标准到标准件的一站式服务，**精度认证齐全**，支持多种材料与工艺。 |
| **[爱合发 (iHF)](https://www.ihf.cn)](https://www.ihfcn.com/)** | **FA自动化零部件**集成与定制平台 | **场景**：自动化设备开发中，标准件选型与**非标件定制无缝衔接**的需求。<br>**特点**：海量FA标准品数据库，同时提供便捷的非标定制入口，生态协同好。 |

## 二、电子与硬件创新一站式平台

服务于硬件创业者、电子工程师，提供从“想法”到“产品”的完整供应链。

| 平台 | 核心定位 | 适用场景与核心特点 |
| :--- | :--- | :--- |
| **[嘉立创](https://www.jlc.com)** | **电子+机械**全产业链一站式平台 | **场景**：电子硬件项目从0到1全流程，如智能硬件、IoT设备开发。<br>**特点**：**“一个概念进，一个产品出”**，集成EDA、PCB打样、元器件商城、SMT贴片、CNC/钣金外壳加工，生态闭环强大。 |
| **捷配** | **PCB打样** | 以“协同制造”模式整合外部工厂，强调灵活性与快速响应，是PCB打样领域最直接的竞争者之一。 |
| **猎板** | **PCB制造（侧重中高端）** | 专注于**高多层板（12层以上）、HDI板及厚铜板**，在通信、军工等对工艺要求更复杂的领域与嘉立创形成差异化竞争。 |
| **云汉芯城** | **元器件电商、一站式供应链** | 定位为电子产业互联网平台，核心优势在于强大的**数字化基建与数据能力**，在**小批量、多品种**的元器件采购领域是主要对手。 |
| **立创商城** | **元器件电商、EDA工具** | 同样是“EDA+元器件+社区”的生态模式，其特点是**自营现货和极速发货**，与嘉立创的商业模式几乎完全重合，是生态闭环内的直接竞品。 |
| **华强电子网** | **元器件贸易（特别是现货/稀缺料）** | 依托华强北现货市场，在**寻找稀缺料号、停产件及应对紧急缺货**方面拥有无与伦比的信息广度和灵活性。 |

## 三、MRO与工业品采购平台

解决企业维护、维修、运营（MRO）及生产物料的采购与管理需求，提升间接物料采购效率。

| 平台 | 核心定位 | 适用场景与核心特点 |
| :--- | :--- | :--- |
| **[怡合达](https://www.yiheda.com)** | **工厂自动化(FA)零部件**一站式供应商 | **场景**：自动化设备设计、组装与维护。<br>**特点**：产品目录化、标准化程度极高，**选型软件与3D模型**齐全，是自动化工程师的“标准件仓库”。 |
| **[震坤行](https://www.zkh.com)** | **全栈式数字化工业品**服务平台 | **场景**：大中型企业MRO及通用生产物料的**集采、管理与降本**。<br>**特点**：SKU海量，提供智能仓储、采购SaaS等数字化工具，强在**企业级供应链解决方案**。 |
| **[易买工品](https://www.e-mro.com)** | **聚焦中小企业的长尾MRO电商** | **场景**：中小企业小额、多频次、急需的MRO采购。<br>**特点**：**价格透明、起订量低、配送快**，用户体验接近消费品电商，性价比高。 |
| **[西域](https://www.ehsy.com)** | **专业MRO解决方案**提供商 | **场景**：对产品质量、合规性、供应链稳定性要求严苛的**大型企业、外资企业**。<br>**特点**：正品保障，提供专业的选型支持与合规方案，强在**BOM采购与流程合规**。 |
| **[鑫方盛](https://www.xinfangsheng.com)** | **工业品生态链**平台 | **场景**：大型工程项目、建筑施工及工厂的综合性物资采购。<br>**特点**：融合工业品与周边消费品，提供一站式的**工地/项目物资解决方案**。 |
| **[工品汇](https://www.vipmro.com)** | 专注于**MRO的供应链**平台 | **场景**：常规MRO物料的稳定补货与采购。<br>**特点**：传统MRO品类齐全，供应链稳定，是许多企业的**基础型采购渠道**。 |
| **[瑞物云](https://www.rewoom.cn)** | **电气行业非标产品**一站式AI平台 | **场景**：数据中心、机房建设中的**配电、UPS等非标电气方案**快速配置与采购。<br>**特点**：利用AI技术快速匹配方案，将复杂非标电气产品**标准化、模块化**输出。 |
| **[中信易家](https://www.citichome.com)** | **集团级智慧采购**平台 | **场景**：大型集团、国企的办公用品、通用物资的**集中化、合规化采购**。<br>**特点**：依托中信集团资源，强在**大客户服务、供应链金融与合规流程**。 |

## 四、跨境与综合外贸服务平台

助力中国制造企业开拓国际市场，提供从营销到交付的全链条外贸服务。

| 平台 | 核心定位 | 适用场景与核心特点 |
| :--- | :--- | :--- |
| **[中国制造网](https://www.made-in-china.com)** | **专注内贸转外贸的B2B**平台 | **场景**：希望将标准品或定制产能推向海外市场的制造工厂。<br>**特点**：**海外买家流量精准**，是传统工厂触达国际客户的经典渠道。 |
| **[中基汽通全球](https://www.cnauto-global.com)** | **汽配产业跨境供应链**平台 | **场景**：汽车零部件制造商与贸易商的**出海业务**。<br>**特点**：提供从产品认证、海外营销、跨境物流到海外仓配的**汽配垂直领域全托管服务**。 |
| **[中建材易单网](https://www.egdchina.com)** | **建材行业跨境综合服务**平台 | **场景**：建材、家居类产品的出口。<br>**特点**：背靠中国建材集团，提供**建材垂直领域的专业外贸、物流、金融及海外仓服务**。 |

## 总结与选择建议

| 需求类型 | 优先考虑平台 |
| :--- | :--- |
| **机械零件快速打样/小批量** | 速加网、Xometry、立谱智造 |
| **高精度金属零件加工** | 铨洲智造、RapidDirect、Fictiv |
| **复杂非标件全球寻源** | 海智在线、Xometry |
| **硬件创业全流程支持** | **嘉立创**（电子+机械闭环） |
| **FA自动化标准件采购** | **怡合达**、爱合发(iHF) |
| **企业MRO数字化集采** | **震坤行**、西域（大型企业）；易买工品（中小企业） |
| **汽配/建材行业出海** | 中基汽通全球、中建材易单网 |
| **通用性外贸推广** | 中国制造网 |

**核心趋势**：平台正从“信息撮合”向“**深度服务与数字化赋能**”演进。AI核价、全流程可视化、供应链金融、生态闭环成为竞争关键。建议企业根据自身**阶段（研发/量产）、品类（标准/非标）、规模（金额/数量）和核心诉求（速度/成本/质量）** 进行组合式选择。
